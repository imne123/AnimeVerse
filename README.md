1. AnimeVerse

AnimeVerse is een interactieve single-page webapplicatie gebouwd met JavaScript en Vite.
Gebruikers kunnen anime zoeken, filteren, sorteren en opslaan als favoriet.


2. Functionaliteiten

- Anime ophalen via API
- Zoekfunctie
- Filter op score
- Sorteerfunctie
- Favorieten opslaan met localStorage
- Responsive design
- IntersectionObserver
